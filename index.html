Project 1 - BlindTest via Spotify API

Figma du projet - Github du projet

Ce projet a été conçu pour s'amuser entre amis lors d'un week-end du Nouvel An 2024-2025. Développé en Java , il s'agit d'un blind test utilisant l'API de Spotify pour récupérer les musiques d'une playlist personnalisée. Actuellement en cours de développement.

Mini-jeux Projet

Ce projet est un exercice personnel visant à créer un mini-jeu inspiré de CodeNames. Il est développé en utilisant les trois langages de base du web : HTML5 , CSS3 et JavaScript . Actuellement en cours de développement, l’objectif est de le déployer sur un serveur afin de le rendre jouable en ligne.
